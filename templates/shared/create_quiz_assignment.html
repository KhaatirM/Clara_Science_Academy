Paste one question per line. Each line becomes one question (short answer by default).

Upload a CSV file. First row must be headers.

Required columns:
  • question_text – the question
  • question_type – one of: multiple_choice, true_false, short_answer, essay
  • points – number (e.g. 1 or 2.5)
Optional columns:
  • section – section/part title (a new section is created when this value changes)
For multiple choice only:
  • option_a, option_b, option_c, option_d – answer choices
  • correct – letter of correct answer: A, B, C, or D
Example (first row = headers): 
section,question_text,question_type,points,option_a,option_b,option_c,option_d,correct
